1. Open terminal on the frontend folder and run "npm install"
2. Open terminal on the backend folder and run "npm install"
3. Having terminal opened on the backend run "npx nodemon server.js" to run the application
4. Having terminal opened on the frontend run "npm start" to run the application
5. Optionally run seeders by running "npm run seed:data" (being in the backend folder)
6. Register account as a user
